# OmniFood

## Project Overview

OmniFood is a responsive landing page for a fictional healthy meal delivery service. It is built using HTML and CSS with a focus on clean design and responsive layouts.

### Technologies Used
- HTML5
- CSS3
- Flexbox
- CSS Grid
- Media Queries

---


## How to Run

1. Clone the repository.
2. Open the project folder.
3. Open `index.html` in your browser.

No installation or dependencies are required.

---

## Responsive Breakpoints Used

- **Desktop:** 1200px and above
- **Tablet:** Up to 944px
- **Small Tablet:** Up to 704px
- **Mobile:** Up to 544px
